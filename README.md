# 3 Year

Welcome to the overview repository for Year 3. This repository contains course materials, project work, and notes for Semester 1 and Semester 2, with a strong focus on Work-Based Learning (WBL) and data engineering.

---

## 📅 Semester 1

### 🧠 Artificial Intelligence
* **Summary:** Explores the foundational concepts and practical applications of AI. Topics include machine learning algorithms, search strategies, neural networks, knowledge representation, and natural language processing. The course emphasizes building intelligent agents and deploying predictive models to solve complex, real-world problems.

### 🗄️ Database Programming
* **Summary:** Focuses on advanced database management concepts and server-side programming. Students delve deep into structured query language (SQL), PL/SQL, stored procedures, triggers, indexing, and query optimization. It covers managing transactional integrity and developing robust database architectures to support large applications.

### 🏢 Enterprise System Design And Modeling (WBL)
* **Summary:** A Work-Based Learning (WBL) module focused on architectural patterns and methodologies for designing large-scale enterprise systems. It covers software architecture styles (microservices, monolithic, event-driven), system modeling techniques (UML, data flow diagrams), scalability, security, and integrating corporate software with existing enterprise infrastructure.

---

## 📅 Semester 2

### 📱 Application Development (WBL)
* **Summary:** A hands-on, Work-Based Learning (WBL) module dedicated to building full-stack applications. It covers modern software engineering practices, agile methodologies, frontend/backend frameworks, API development, and automated deployment pipelines, ensuring projects meet industry-standard production requirements.

### ⚡ High Performance Data Processing
* **Summary:** Focuses on the architectures and frameworks required to process massive datasets efficiently. Students learn about parallel computing, distributed systems, stream processing, and big data technologies (such as Apache Spark and Hadoop) to build high-throughput, low-latency data pipelines.

### 📊 Business Intelligence
* **Summary:** Covers the tools and methodologies used to transform raw data into actionable business insights. Key topics include data warehousing, ETL (Extract, Transform, Load) architecture, multi-dimensional data modeling, interactive dashboards, and reporting tools that assist executive decision-making.

### 🚀 Special Topic In Data Engineering
* **Summary:** An advanced module exploring cutting-edge developments, emerging trends, and specialized technologies in modern data engineering. Topics vary to match current industry demands, potentially covering cloud data architectures, advanced data governance, specialized MLOps pipelines, or decentralized data webs.
